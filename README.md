# PuffingAndBlowing
프로젝트 명: 허둥지둥 Puffing And Blowing<br>
프로젝트 내용: 유니티로 개발한 모바일 앱입니다. 화면의 한 부분을 터치 시 캐릭터가 몹시 느린 속도로 허둥지둥 터치 지점을 향해 움직이게 됩니다. 터치 지점에 도달한 캐릭터는 모습이 바뀝니다!<br>
